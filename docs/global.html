combat(jugador, enemigo)

Ejecuta un combate entre un jugador y un enemigo, simulando turnos hasta que uno de los dos llegue a 0 puntos de vida o menos. Calcula el daño total recibido por el jugador, los puntos obtenidos por la victoria (si la hay) y actualiza la vida y puntos del jugador. Finalmente muestra en pantalla el ganador y los puntos conseguidos.

groupBy(array, keyFunc) → {Record.<string, Array.<any>>}

Agrupa los elementos de un array según una función clave que se pasa como parámetro. Recorre el array y aplica `keyFunc` a cada elemento para obtener la clave de agrupación. Devuelve un objeto cuyas propiedades son las claves (String) y cuyos valores son arrays con los elementos que pertenecen a cada grupo.
Parameters:
Name Type Description
array Array.<any> Array de elementos a agrupar.
keyFunc function Función que recibe un elemento y devuelve la clave por la que se agrupará.
Source:
  • Utils/utils.js, line 68
Returns:
Objeto que mapea cada clave a un array con los elementos del grupo.
Type
