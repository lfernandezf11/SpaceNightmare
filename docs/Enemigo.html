Class: Enemigo

Enemigo(tipo, nombre, ataque, vida, avatar)

Representa a un enemigo genérico del juego. Se usa como base para cualquier contrincante que pueda entrar en combate. Contiene estadísticas comunes como ataque, vida y un avatar para la UI.
